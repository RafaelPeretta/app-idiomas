# app-idiomas
Repositório referente ao projeto do Passaporte Universitário da Faculdade SENAI Félix Guisard - SESI São Paulo.

Este projeto consiste no desenvolvimento de um aplicativo introdutório de ensino de idiomas que tem como objetivo auxiliar os usuários na aprendizagem de alfabetos e estruturas básicas, oferecendo uma base sólida para estudos mais avançados. A ideia é tornar o aprendizado de línguas estrangeiras mais acessível e intuitivo em um mundo globalizado.
O aplicativo é uma ferramenta pedagógica de apoio para estudantes e futuros profissionais, especialmente no contexto do SESI, que tem uma forte ligação com a indústria. Dominar um segundo idioma, mesmo que em nível básico, pode abrir melhores oportunidades.

🎯 Objetivos

O principal objetivo é desenvolver um aplicativo introdutório de ensino de idiomas que auxilie o usuário na aprendizagem de alfabetos e estruturas básicas, promovendo uma base sólida para estudos mais avançados. Para alcançar este objetivo geral, a equipe se dedicará a criar uma interface interativa e acessível para iniciantes em diferentes idiomas. Serão implementados módulos para o ensino de alfabetos, pronúncias e estruturas básicas, visando uma introdução completa. Por fim, será fundamental testar e validar a eficácia do aplicativo com usuários em ambiente escolar do SESI, garantindo que ele cumpra seu propósito de forma eficiente.

🛠️ Ferramentas e Tecnologias

O desenvolvimento deste aplicativo é fundamentado em um conjunto de tecnologias robustas para garantir um produto funcional e de alta qualidade. A equipe utilizou o Unity Engine UI para a interface do usuário, permitindo a criação de uma experiência visual e interativa. A principal linguagem de programação utilizada para as funcionalidades do aplicativo é o C#. Além disso, o Firebase foi escolhido para gerenciar o banco de dados e o sistema de autenticação, o que inclui o login e o registro dos usuários. Os desenvolvedores também precisam ter acesso a um computador e à internet para realizar o projeto.
